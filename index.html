INSTRUCCIONES
1. Creá un repo en GitHub (ej: papyrus-data) — puede ser privado.
2. Generá un token en github.com → Settings → Developer settings → Personal access tokens → Fine-grained.
3. Dale permisos: Contents: Read and Write en ese repo.
4. Pegá los datos abajo. Se guardan solo en este navegador.
Se guarda solo en localStorage de este navegador. No viaja a ningún servidor.
Dejá db.json salvo que tengas una razón específica para cambiar.
